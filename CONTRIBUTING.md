# Contributing

This repository is currently an architectural experiment.

---

## Guidelines

- Keep Version 1 simple.
- Do not add database logic to V1.
- Do not introduce authentication providers in V1.
- Preserve authority-based structure.
- Maintain type safety.

---

## Branching Strategy

- main → Stable releases
- dev → Active development
- feature/* → Experimental features

---

## Version Philosophy

V1 → Controlled Lab  
V2 → Structured Permission Engine  
V3 → Production Architecture
